# nvim
My Neovim Config

Requirements:
- [ripgrep](https://github.com/BurntSushi/ripgrep)
- A terminal that supports true color and undercurl:
    - [wezterm](https://github.com/wez/wezterm) (Linux, Macos & Windows)
- Requires pylint 
- Requires npm for treesitter
- Requires c compiler
